## 使用方法

```typescript

let result = await fccInstance.connect({
  path: '/xw' // 对应服务端上下文
})

```
<!-- **入参说明** -->
### 入参说明

| **参数名** | **数据类型** | **选取原则** |**说明** |
| ---------- | ------------ | ------------ | ------------------ |
| path      | string       | 可选         | 连接中台服务的上下文，默认'/' |

### 出参说明

| **出参名称** | **数据类型** | **说明**                         |
| -------- | -------- | ------------------------------ |
| status   | number   | 调用接口返回结果的状态码，其含义参考融合通讯中台通用状态码。 |
| msg      | string   | 给开发者的文字提示信息                    |
| data     | string   | 返回的信息                          |
