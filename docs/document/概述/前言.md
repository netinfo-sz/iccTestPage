# 前言
## 概述
融合通讯中台提供纯 `javascript` 的中台服务，供业务封装使用

可以给第三方系统提供底层点对点音视频通话、视频会议、电话接报、电话外呼、实时浏览监控、IM即时聊天的功能

第三方系统通过WEB前端引用中台 `javascript` ，进行二次开发。融合通讯中台 `js-api` 均采用主流的 `Promise` 风格封装，开发者通过简单调用js方法、监听js事件方便快速地开发出优秀的融合通讯指挥调度平台
